# IMDF Mobile — Téléchargements

Dépôt public de **distribution** de l'application mobile IMDF.

> ⚠️ Ce dépôt ne contient **aucun code source**. Il sert uniquement à héberger les binaires de l'application (APK Android, IPA iOS) publiés automatiquement via GitHub Actions.

## Télécharger l'application

Rendez-vous dans l'onglet **[Releases](../../releases)** pour récupérer la dernière version :

- **Android** : fichier `.apk` — téléchargez-le et installez-le directement sur votre appareil (autorisez les sources inconnues).
- **iOS** : fichier `.ipa` — nécessite une installation via un canal signé (TestFlight ou MDM).

Chaque release indique la version, la date de build et l'environnement (staging / production).
